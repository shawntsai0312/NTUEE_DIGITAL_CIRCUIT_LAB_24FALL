# Lab 3 數位錄音機

### Before Running

```shell
cd Lab3/sim/
source tool.sh
```

### How To Run Simulation

1. run the following commands in the terminal
    ```shell
    cd Lab3/sim/sh/[MODULE_NAME]
    source run.sh
    nWave &
    ```

2. open nWave and open the file `sim/sh/tb_MODULE_NAME.fsdb`
3. select desired signals
